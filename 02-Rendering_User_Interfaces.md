# Chapter 2: Rendering User Interfaces (UI)

## The DOM (Document Object Model)

### What is the DOM?
The DOM is an object representation of HTML elements that acts as a bridge between your code and the user interface.

**Structure**: Tree-like with parent and child relationships

### How It Works
1. Server returns HTML file to browser
2. Browser reads HTML and constructs the DOM
3. DOM represents the page as objects that can be manipulated

## DOM Manipulation

You can use DOM methods and JavaScript to:
- Listen to user events
- Select specific elements
- Add, update, and delete elements
- Change element styles and content

## Key Takeaway
Understanding the DOM is fundamental to understanding how React works, as React provides a more efficient way to update the DOM.
