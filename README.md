## Key Concepts

**React**: JavaScript library for building interactive UIs (provides tools)
**Next.js**: React framework that adds structure, routing, and optimizations (provides structure + tools)

**Core React Concepts**:
- **Components**: Reusable UI building blocks (functions that return JSX)
- **Props**: Data passed from parent to child (read-only, one-way flow)
- **State**: Data managed within a component (mutable, triggers re-renders)

**Programming Paradigms**:
- **Imperative**: Step-by-step instructions (HOW to do something)
- **Declarative**: Describe desired outcome (WHAT you want)

**JSX**: HTML-like syntax in JavaScript, compiled by Babel to regular JavaScript
